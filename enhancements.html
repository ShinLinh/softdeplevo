  1. Figure and Figcaption html tags:

    The <figure> tag is used to mark up self-contained elements like photos, so it could be used for the picture of the student. It is useful for the assignment design because of the <figcaption> tag that is used to create captions for the element in the <figure> tag. This will help us avoid having to write separate codes to position the caption.

  2. Creating link that help jump to different parts of the web page:

    It is possible to create a link to jump to the position with the <a id="name"> tag on a web page with a link using the tag <a href="#name"> . This function can help the user instantly move to a specific section of the web page.

  3. Embedding a video on the web page

    We can use the <iframe> tag to link to a video from another web page and put it on our own. This saves loading time for the user by letting them watch a video from a different source without opening another page.
